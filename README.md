## 简介
---
本项目是一个基于AutoGluon开发的COPD病人院内死亡概率及住院时长的预测模型。

## 环境
```Bash
pip install -U pip
pip install -U setuptools wheel

pip install autogluon

```

## 目录说明
---
LOSPrediction.py可以依据病例中的文本信息实现对COPD病人住院时长的预测  

MortalityPrediction.py可以根据表格化的检查检验数据实现对COPD病人院内死亡率的预测
